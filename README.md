# Assignment-3
Game Idea

Find a hidden treasure in a 5×5 grid by guessing the row and column.

Rules

Enter row and column (1–5).

Invalid inputs are rejected.

Attempts increase with every valid guess.

Game ends when you hit the correct coordinates.


Controls

Input row (1–5)

Input column (1–5)


Features

Random treasure location

Direction hints (UP, DOWN, LEFT, RIGHT)

Unlimited attempts

Simple and beginner-friendly
